# Project Showcase

## Purpose
Showcase projects Daniele has built — demonstrating real-world application of his expertise.

## Requirements (Initial)
- Each project should have: name, description, technologies used, link (if applicable), and possibly a screenshot or visual
- Projects should be filterable or categorized
- Animated reveal on scroll or interaction
- Styled consistently with the overall "high-tech" aesthetic — e.g., project cards as "system modules" or "classified files"
