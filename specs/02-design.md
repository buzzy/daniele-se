# Design & Aesthetic

## Visual Direction
"High-tech server OS" — inspired by sci-fi movie interfaces, terminal UIs, and hacker aesthetics. The site should feel like an operating system or command-line environment come to life in the browser.

## Aesthetic Elements (Initial)
- Dark background (near-black) with high-contrast accent colors (e.g., green, cyan, amber terminals)
- Monospace / terminal fonts
- Animated elements: typing effects, scanlines, glitch effects, matrix-style animations
- Grid/HUD overlays reminiscent of military or sci-fi dashboards
- Boot/loading sequences on first visit
- Possibly: ASCII art, command-line navigation metaphors

## Tone
- Confident, technical, impressive
- No fluff — every element serves a purpose or communicates expertise
- Interactive where possible — the user should feel like they are operating something

## Sections Design Notes
- **Hero:** Full-screen immersive intro — possibly a simulated terminal boot sequence
- **Skills:** Visual "system readout" — technologies displayed as processes, modules, or system specs
- **Projects:** Card-based or panel-based showcase with animated reveals
- **Contact:** Clean, minimal — possibly styled as a terminal prompt

## Responsiveness
Must work on desktop and mobile, though the primary experience is desktop-first given the aesthetic.
